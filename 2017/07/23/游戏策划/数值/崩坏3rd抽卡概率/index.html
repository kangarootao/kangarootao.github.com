崩坏3rd抽卡概率研究

发表于 2017-07-23   |   分类于 数值研究   |  

研究的起点是游戏中精准补给的概率，四星武器概率4.958%，四星圣痕概率7.437%，当期up武器概率2.479%，圣痕每个1.240%，而保底是保底四星（大奖），且是在出过四星后重置保底所需数量。这概率不像一般其他游戏显示的是1%，1.5%等较整的数而是一个精确的小数，这令人费解，而且体感上来说并没有这么高的单次出率。而扩充和标配则是1.5%，1.33%出s级女武神（即大奖），是一个较整的数，这两者的区别在于保底出的是否为当期大奖。那么，精准补给的概率是否是加入保底后进行计算的呢？仔细研究说明后发现并没有说明，以下为试验性计算。

显示概率：
up武器：2.479%
up单圣痕：1.240%
四星：4.958% + 7.437% = 12.395%
假设保底时概率分布与显示概率一致。
假设显示概率是加入保底计算的单次抽卡的期望概率。

1.保底策略：连续十次抽卡必定至少有一个四星，即出了四星后重置计数器。
设单次抽取的真实概率：
up武器：y
四星：x
x = 5 * y
抽到一个四星的平均次数：
m = 1 * x + 2 * x * (1 - x) + 3 * x * (1 - x) ^ 2 + … + 9 * x * (1 - x) ^ 8 + 10 * (1 - x) ^ 9
m = 1 / 12.395% = 8.06777
九次方程，不好解。但是x取值范围为[0,1]，所以可以用程序暴力搜索求近似值。
python程序：
x = 0
step = 0.00001
mmin = 99999

def func(x, n):
  result = 0.0
  for i in range(0, n-1):
    result += (i + 1) * x * (1 - x) ^ i
  result += n * (1 - x) ^ (n - 1)
  return result

while(x<1):
  x += step
  result = func(x, 10)
  if abs(result - 8.06777) < mmin:
    mmin = abs(result - 8.06777)
    ans = x

print(ans)

运行结果为：0.04877
也就是说在没有保底的情况下，单抽出四星的概率为4.88%，比较符合真实情况。推断：显示概率为期望概率。

2.保底策略：十连抽必定至少有一个四星，每次十连独立计算。
设单次抽取的真实概率：
up武器：y
四星：x
x = 5 * y

进行一次十连，无保底时，抽到四星的期望个数：
10 * x
连续九次都没抽到四星的概率：
(1 - x) ^ 9
加入保底后，这部分概率产生的四星个数由0变成1，四星期望个数增加：
(1 - x) ^ 9
所以加入保底后抽到四星的期望个数：
n = 10 * x + (1 - x) ^ 9 = 10 * 12.395%

同样使用程序解方程，结果为0.07378

总结：
为了达到12.395%的预期概率，第一种保底策略的单次抽取真实概率为4.88%，第二种保底策略的单次抽取真实概率为7.38%，第一种保底策略较第二种提高了不少的预期出率，而崩坏3rd属于少见的使用第一种保底策略的。在游戏中，卡池说明单次抽取四星概率为12.395%与4.88%都是没有问题的，而前者将给予玩家“出率还挺高，我再抽抽就出了”的错觉。并且，一次抽取开两箱的设定也给予玩家一次十连相当于二十连的错觉（刚入坑我也是被这么骗的），能够很大程度上吸引玩家继续抽卡。
使用预期概率另一个好处在于，计算期望花费十分方便，不需要考虑保底的问题。

例如强怼一把冰昙天。
648rmb = 8088钻 = 28.89抽
1抽 = 22.43rmb
up武器期望概率为2.479%
期望花费：
22.43 / 2.479% = 904.8rmb

例如强怼一套浓姬圣痕。
单up圣痕期望概率为1.240%
期望花费：
22.43 / (1.240% * 3) + 22.43 / (1.240% * 2) + 22.43 / 1.240% = 3316.3rmb

比起抽角色来还是简单不少的。也说明了新的精准补给的收入一般是不如扩充补给的，除非什么好东西都塞进去了，不过会减少游戏寿命。而且扩充还能抽到SSS，对于微氪也算是半个无底洞吧。（理性氪金= =）
