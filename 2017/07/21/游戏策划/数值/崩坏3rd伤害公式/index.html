崩坏3rd伤害公式

发表于 2017-07-21   |   分类于 数值研究   |  

根据论坛的实验结论：
元素伤害 = (攻击力系数+被动元素伤害)(1+元素伤害Buff)(1+全伤害Buff)(1+减伤+Debuff+属性克制)
物理伤害 = (攻击力系数)(1+物理伤害Buff)(1+全伤害Buff)(1+减伤+防御力减伤+Debuff+属性克制)
暴击率=会心/(75+5*角色等级)
属性克制系数为+-0.3
精英护盾减伤为20%
Boss护盾减伤为50%

例如计算本萌新的勿忘的单发斩击伤害：
面板攻击682，会心72
武器lv35冰昙天，提升7.0%元素伤害，主动增加冰冻元素伤害16%
圣痕浓姬套。lv45上提升对冻结敌人全伤害33%。lv35中攻击附带50%攻击力的冰冻元素伤害，1s冷却。lv35下每有一个移速降低或冻结的敌人提升15%冰冻元素伤害，最多三层。套装提升冰冻元素伤害80%。
使用必杀技后全伤害提升40%。
被动增加对眩晕，麻痹，冻结，时空减速敌人的物理和元素伤害35.9%。
强化斩击造成100%攻击力的物理伤害+500%攻击力的冰冻元素伤害。

0Buff无克制情况：
物理伤害 = 682111 = 682
暴击率 = 72/(75+565)= 18%
期望物理伤害 = 6821.18 = 804.8
冰冻元素伤害 = 6825(1+0.07+0.8)1*1 = 6376.7

3怪减速无克制情况：
物理伤害不变
冰冻元素伤害 = 6825(1+0.07+0.8+0.45)11 = 7911.2

3怪冻结无克制情况：
物理伤害 = 682(1+0.359)(1+0.33)1 = 1232.7
冰冻元素伤害 = 6825(1+0.07+0.8+0.45+0.359)(1+0.33)*1 = 12150

必杀技后3怪冻结开武器主动克制情况：
物理伤害 = 682(1+0.359)(1+0.33+0.4)(1+0.3) = 2084.5
冰冻元素伤害 = 6825(1+0.07+0.8+0.45+0.359+0.16)(1+0.33+0.4)*(1+0.3) = 21772.5
