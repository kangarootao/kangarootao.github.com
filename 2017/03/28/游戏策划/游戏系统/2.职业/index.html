职业系统

发表于 2017-03-28   |   分类于 游戏系统   |  

系统目标
1.对玩家或角色的职能进行区分，不同职业拥有的功能不同，例如输出、坦克、治疗等。
2.提高角色多样性，不同职业以及搭配能给予多样的新鲜感。
3.可以设置职业克制，在一定条件下筛选出“最优”选择。

实现方法
1.设置不同职业供玩家选择。
2.达到要求时可进行下一步职业进阶（二转）。
3.特定条件下可更换职业。
4.特定游戏中可以同时拥有多职业。

种类
按功能分类：
输出：
1.根据攻击距离不同，分为远程输出与近战输出。
2.根据输出方式不同，分为物理输出与魔法输出。
3.根据输出属性不同，分为各类属性攻击，如火、冰、雷。
4.根据输出时能否兼顾走位，分为站桩输出与移动输出。
5.根据输出目标的多少，分为单体输出与群体输出。
6.根据输出的生效时间，分为瞬时输出与滞后输出。

治疗：
1.根据治疗生效时间，分为即时治疗、持续治疗与预先治疗。
2.根据治疗目标的多少，分为单体治疗与群体治疗。
3.根据治疗时能否兼顾走位，分为站桩治疗与移动治疗。

坦克：
1.根据防御方式不同，分为物抗坦克、魔抗坦克、闪避坦克（还有格挡、招架等）。
2.根据能否自我治疗，分为自疗坦克与非自疗坦克。
3.根据属性倾向不同，分为护甲型（高减伤）与血量型（高血量）。

辅助：
1.根据辅助技能的性能不同，分为增强辅助与削弱辅助。
2.根据目标数量不同，分为自身、单体、群体辅助。
3.根据辅助技能种类不同，分为硬控（受控者无法操作，如晕眩、睡眠）、软控（受控者可以操作但受限，如减速、致盲）、削弱（降低属性）、战斗增益（增强属性）、非战斗增益（传送、造食）。

职业倾向可以由以上不同的功能进行组合后得出。以WoW为例，冰法属于远程、魔法、冰系、半站桩半移动、单体倾向的爆发输出，同时拥有预先治疗手段（冰盾），擅长对敌方单体与群体进行硬控与软控（深度冻结与冰霜新星）。以WOT为例，重坦一般属于较近战、站桩、单体瞬时输出，兼具护甲型坦克的属性。
